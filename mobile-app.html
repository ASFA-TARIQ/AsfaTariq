Mirror of Magic

A magical quiz that reveals the Disney character within you.

Project Overview

Mirror of Magic is an interactive personality-style mobile application designed to provide users with a playful and nostalgic Disney-inspired experience. The application allows users to discover a Disney character that matches their preferences by selecting different options such as colour, activity, and destination.

The goal of the project was to design a fun and engaging digital experience that combines personality quizzes with interactive visual design. By guiding users through a short series of choices, the application generates a unique result that introduces a Disney character along with a short description that reflects the user’s selected preferences.

The app was designed to be simple, visually appealing, and accessible for users of all ages. Through a combination of animation, themed design, and interactive elements, the application creates an enjoyable and nostalgic experience inspired by classic Disney storytelling.

Target Audience

The application is designed for a broad audience including children, teenagers, and adults who enjoy Disney characters and interactive quizzes. It is especially suitable for users who want a quick and entertaining digital experience.

The interface is intentionally designed to be simple and visually guided so that even younger users can easily navigate through the app without difficulty. The combination of images, text, and interactive buttons makes the experience both accessible and enjoyable.


Design and Layout Decisions

The application interface was designed using Figma, which was used to create the wireframes and interactive prototype of the app. The design process focused on building a clear and intuitive user journey across the application's six main screens: Splash Screen, Home Screen, Choose Colour Page, Choose Activity Page, Choose Destination Page, and the Result Page. Each screen was carefully structured to maintain visual consistency and ensure smooth navigation throughout the user experience.

Visual elements such as buttons, icons, and background graphics were designed using Canva. The overall colour theme follows a pastel blue Disney-inspired style, helping to create a magical and nostalgic atmosphere that aligns with the concept of discovering Disney characters.

Typography was also an important part of the design process. Two main fonts were selected to strengthen the visual identity of the application. Tangerine was used for headings to create an elegant and magical look, while Cinzel Decoration was used for category elements to enhance the thematic style of the interface.

To make the interface accessible for users of all ages, especially younger users, each option was designed using both images and text. This visual approach allows users to understand the options quickly, even without reading detailed instructions.

To further enhance engagement, button animations were added to create a more dynamic and playful interaction. These animations support the magical Disney theme and make the experience feel more interactive.

Additionally, background music inspired by Disney movies was integrated into the application to strengthen the atmosphere and provide a more immersive user experience.

Wireframes

Before building the application, wireframes were created to plan the structure and layout of the interface. Wireframes helped define the placement of elements such as navigation buttons, text, images, and user interactions across the different pages of the app. They allowed the design to be refined before the final interface was developed.

Actual Interface Design

After finalizing the design, the application interface was implemented based on the wireframe structure. The final app includes animated buttons, themed backgrounds, character images, and interactive pages that guide the user through the selection process. The result page dynamically displays a Disney character with a matching description based on the user’s selected preferences.

Critical Reflection

Developing the Mirror of Magic application helped me gain practical experience in designing an interactive and user-friendly application. The project allowed me to apply principles of user interface design, prototyping, and user-centered thinking while creating a fun personality-style app. By guiding users through a simple selection process, the app provides a personalised result, making the experience engaging and easy to understand for users of different age groups.

One of the main strengths of the application is its clear structure and intuitive navigation. The use of both images and text makes it easier for users, especially younger audiences, to quickly understand the options and interact with the app. Additionally, the inclusion of animated buttons and themed visual elements enhances the overall experience by making the interface feel playful and aligned with the Disney-inspired theme.

However, the project also highlighted areas for improvement. For example, the background music looping issue slightly disrupts the user experience when the audio restarts. In the future, the application could be improved by adding more selection categories, multilingual support, and optional text-to-speech functionality so users can listen to their results. These enhancements would increase accessibility and provide users with a richer and more personalised experience.
